OpenPOS is an open source, cloud-ready Point-of-Sale system for small businesses and restaurants. Built on the MEAN stack (MongoDB, Express, AngularJS, Node.js), it gives you a fully browser-based POS interface — no native app required. Register an account, populate your menu, and start taking orders in minutes.

## How it works

OpenPOS runs as a Node.js web server you access through any modern browser. Each user account has its own isolated product catalog stored in MongoDB. The AngularJS frontend communicates with the Express backend over a simple REST API to fetch, add, and remove products.

<Steps>
  <Step title="Create an account">
    Register with a name, email, username, and password. Your credentials are stored securely with bcrypt hashing.
  </Step>
  <Step title="Build your menu">
    Add menu items with names, prices, and categories (Foods, Drinks, Other) using the **Edit Items** tab in the POS interface.
  </Step>
  <Step title="Take orders">
    Click items in the menu panel to add them to the live order summary. Adjust quantities or remove items before checkout.
  </Step>
  <Step title="Process checkout">
    Hit **Checkout** to complete the order. OpenPOS displays the order total and increments your session order count.
  </Step>
</Steps>
